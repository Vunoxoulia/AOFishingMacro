How to Set Up Keybinds

Enter key combinations in the text fields below. Use the format: Modifier + Key (e.g., "Shift+A"). Modifiers are optional for single keys.

  • Supported Modifiers (case-insensitive): Shift, Ctrl/Control, Alt, Win.
  • Keys: Letters (A-Z), numbers (0-9), F-keys (F1-F12), or special keys like Space, Tab, Enter.
  • Examples:
    • "F1" ‚Üí Press F1 alone.
    • "Shift+A" or "shift+a" ‚Üí Press Shift + A.
    • "Ctrl+F6" ‚Üí Press Ctrl + F6.
    • "Alt+1" ‚Üí Press Alt + 1.
    • "Win+Space" ‚Üí Press Windows key + Space.

Tips:

  • Changes save automatically and take effect immediately.
  • If a keybind doesn't work, check the Debug.log file for errors (e.g., invalid key).
  • Avoid conflicting keys (e.g., don't use F1 if it's already bound elsewhere).
  • Single keys like "A" work, but modifiers make them unique.

Note: Multi-key sequences (e.g., "A then B") are not supported. Only modifier + single key combos.
